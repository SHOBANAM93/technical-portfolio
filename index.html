Distributed Topic Crawler

Built a distributed topic crawler, using distributed architecture on elastic search, using Hamachi. The code is written in python, and it was enhanced using advanced data structures and algorithms to crawl 70k documents in 33 hours. The crawled documents were indexed, merged and stored in an Elasticsearch(ES) instance. In addition to this, an angular UI was developed to key in queries, to fetch and view results in real-time from ES. Furthermore, a treceval function was developed to evaluate the accuracy of documents fetched for the queries and was run after doing manual assessments on documents. The interface for the same was added as a feature to the UI.

HR Automation System

The project performs text processing and sentiment analysis on company reviews fetched from web using BeautifulSoup to infer employee sentiment derived from the employee review for the company. The UI for the project was developed using JavaScript, the sentiment analysis of the scraped employee review was done in Python.

Exploratory Data Analysis - U.S Labor Statistics Dataset from 2006-2016

Analyzed and did predictive modelling of U.S Labor Statistics data, on U.S political decisions, poverty and inflation per state.
